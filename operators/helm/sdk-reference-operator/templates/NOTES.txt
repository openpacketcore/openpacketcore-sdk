SDK Reference Operator installed.

1. Verify the controller pod is running:
   kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "sdk-reference-operator.name" . }}

2. Check operator logs:
   kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "sdk-reference-operator.name" . }} -c manager

3. Verify CRDs:
   kubectl get crd sdkmanagednetworkfunctions.reference.openpacketcore.io

4. Create a sample CR:
   kubectl apply -f - <<EOF
   apiVersion: reference.openpacketcore.io/v1beta1
   kind: SdkManagedNetworkFunction
   metadata:
     name: sample-cnf
   spec:
     runtimeMode: dev
     version: "1.0.0"
     configBackend: sqlite
     sessionBackend: fake
   EOF

5. Check status:
   kubectl get sdkmanagednetworkfunctions.reference.openpacketcore.io -n {{ .Release.Namespace }}
